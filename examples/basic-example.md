# Basic Example

> Generate three credible titles from a single content idea.

## What You Will Learn

How to run the skill with one required input and get immediately usable title options.

## Step 1: Run the Skill

```bash
./scripts/run.sh --input "remote onboarding for distributed engineering teams" --count 3
```

## Step 2: See the Output

```text
1. Why remote onboarding for distributed engineering teams deserves a closer look
2. remote onboarding for distributed engineering teams: practical strategies that actually work
3. What teams get wrong about remote onboarding for distributed engineering teams
```
